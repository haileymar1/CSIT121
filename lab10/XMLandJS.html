XML and JavaScript

1. What is XML?

XML stands for Extensible Markup Language. It is a language for storing and transporting data in a structured format. It is a markup language similar to HTML but is meant to store data rather than display it.

2. How is XML used?

XML is used for multiple purposes, such as:

  • Storing and transporting data between applications and systems
  • Storing and exchanging data in a standardized format
  • Defining custom tags and structures for data

3. What does XML look like? (provide an example)

Example of XML code:

<?xml version="1.0" encoding="UTF-8"?>
<book>
  <title>Insurgent</title>
  <author>Veronica Ruth</author>
  <year>2012</year>
</book>

4. What is JavaScript?

JavaScript is a programming language used to create dynamic and interactive web pages. It is also used to add functionality to HTML and CSS, including:

  • Adding interactivity to web pages
  • Creating and manipulating HTML and CSS content
  • Performing calculations and data processing
  • Validating user input

5. How is JavaScript related to HTML and CSS?

HTML, CSS, and JavaScript are all used to create web pages. HTML is used to structure the content of a page, CSS is used to style the content, and JavaScript is used to enhance interactivity and functionality of the page.

6. What are some common JavaScript elements, properties, methods, and functions? (provide at least 1 example of each)

  • Elements: Variables, Experssions, Arrays, Objects
  • Properties: Length, classList, Value
  • Methods: querySelector(), appendChild(), slice()
  • Functions: eval(), parseint(), parseFloat()
